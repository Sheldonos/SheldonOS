## 1. The Signal: Reddit-Sourced Problem

**Subreddit:** r/Entrepreneur

**Original Post:**
> "I have 3 rental units. I manage maintenance requests via text messages and a Google Sheet. Every property management app I've tried starts at $100/month and is built for companies with 50+ units. Small landlords are using texts and prayer."

**Why This Problem Matters:**

There are approximately 17 million individual landlord investors in the United States, the majority of whom own fewer than 5 units. This is not a niche — it is the dominant form of rental property ownership in the country. Yet every property management software product on the market is priced and designed for professional management companies. The gap between "enterprise property management suite" and "Google Sheet + text messages" is enormous, and it is where 17 million landlords currently live.

**Target User:** Independent landlords managing 1–5 rental units who cannot afford or justify enterprise property management software.

**Why Existing Solutions Fail:** Buildium starts at $55/month and requires a minimum of 150 units on its growth plan. AppFolio requires a minimum of 50 units. Propertyware is enterprise-only. The tools built for small landlords (Cozy, TurboTenant) focus on rent collection and tenant screening — not maintenance request management. The maintenance workflow for a small landlord remains: receive a text, write it on a sticky note, call a contractor, forget to follow up.

---

## 2. The Solution: FixIt

**Core Value Proposition:** FixIt gives independent landlords a simple, tenant-facing maintenance request portal with automatic contractor assignment and real-time status updates — replacing the text message and spreadsheet workflow entirely.

**The WOW Feature:** A tenant submits a maintenance request with a photo from their phone using a simple link (no app download, no login required). The landlord sees it instantly on a clean dashboard, assigns it to a contractor with one click, and the tenant receives an automatic status update. The entire workflow — from tenant report to contractor assignment — takes under 60 seconds.

**Supporting Features:** A complete ticket history with status timeline for every property, providing a defensible audit trail for security deposit disputes. A contractor directory linked to the landlord's account, with automatic email notification on assignment. A public-token-based tenant status page that lets tenants check their request status without calling the landlord.

## 4. Generated Artifacts

### 4.1 PostgreSQL Schema (`schema.sql`)

```sql
-- FixIt: Micro-Landlord Maintenance Portal
-- Database Schema — Generated by IBM Project Bob

CREATE TYPE ticket_status AS ENUM ('open', 'assigned', 'in_progress', 'resolved');
CREATE TYPE ticket_priority AS ENUM ('low', 'medium', 'high', 'emergency');

CREATE TABLE landlords (
  id SERIAL PRIMARY KEY,
  name VARCHAR(255) NOT NULL,
  email VARCHAR(255) UNIQUE NOT NULL,
  password_hash VARCHAR(255) NOT NULL,
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE properties (
  id SERIAL PRIMARY KEY,
  landlord_id INTEGER NOT NULL REFERENCES landlords(id) ON DELETE CASCADE,
  address TEXT NOT NULL,
  unit_count INTEGER DEFAULT 1 CHECK (unit_count >= 1),
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE tenants (
  id SERIAL PRIMARY KEY,
  property_id INTEGER NOT NULL REFERENCES properties(id) ON DELETE CASCADE,
  name VARCHAR(255) NOT NULL,
  email VARCHAR(255) NOT NULL,
  phone VARCHAR(20),
  lease_start DATE,
  lease_end DATE,
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE contractors (
  id SERIAL PRIMARY KEY,
  landlord_id INTEGER NOT NULL REFERENCES landlords(id) ON DELETE CASCADE,
  name VARCHAR(255) NOT NULL,
  email VARCHAR(255),
  phone VARCHAR(20),
  specialty VARCHAR(100),
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE maintenance_tickets (
  id SERIAL PRIMARY KEY,
  tenant_id INTEGER REFERENCES tenants(id),
  property_id INTEGER NOT NULL REFERENCES properties(id),
  contractor_id INTEGER REFERENCES contractors(id),
  title VARCHAR(255) NOT NULL,
  description TEXT,
  photo_url TEXT,
  status ticket_status NOT NULL DEFAULT 'open',
  priority ticket_priority NOT NULL DEFAULT 'medium',
  public_token VARCHAR(64) UNIQUE NOT NULL,
  created_at TIMESTAMP DEFAULT NOW(),
  resolved_at TIMESTAMP,
  CONSTRAINT resolved_at_requires_resolved_status
    CHECK (resolved_at IS NULL OR status = 'resolved')
);

CREATE INDEX idx_tickets_status ON maintenance_tickets(status);
CREATE INDEX idx_tickets_property ON maintenance_tickets(property_id);
CREATE INDEX idx_tickets_priority ON maintenance_tickets(priority);
CREATE INDEX idx_tickets_created ON maintenance_tickets(created_at DESC);
```

### 4.2 Core Backend (`server.js`)

```javascript
// FixIt — Core Backend
// Generated by IBM Project Bob

const express = require('express');
const { Pool } = require('pg');
const jwt = require('jsonwebtoken');
const bcrypt = require('bcrypt');
const crypto = require('crypto');

const app = express();
app.use(express.json());

const pool = new Pool({ connectionString: process.env.DATABASE_URL });

const log = (level, message, meta = {}) => {
  console.log(JSON.stringify({ timestamp: new Date().toISOString(), level, message, ...meta }));
};

// Auth middleware
const authenticate = (req, res, next) => {
  const token = req.headers.authorization?.split(' ')[1];
  if (!token) return res.status(401).json({ error: 'Unauthorized' });
  try {
    req.landlord = jwt.verify(token, process.env.JWT_SECRET);
    next();
  } catch {
    res.status(401).json({ error: 'Invalid token' });
  }
};

// GET /api/tickets — All tickets for authenticated landlord
app.get('/api/tickets', authenticate, async (req, res) => {
  try {
    const { rows } = await pool.query(`
      SELECT t.*, p.address, ten.name AS tenant_name, c.name AS contractor_name
      FROM maintenance_tickets t
      JOIN properties p ON t.property_id = p.id
      LEFT JOIN tenants ten ON t.tenant_id = ten.id
      LEFT JOIN contractors c ON t.contractor_id = c.id
      WHERE p.landlord_id = $1
      ORDER BY 
        CASE t.priority WHEN 'emergency' THEN 1 WHEN 'high' THEN 2 
                        WHEN 'medium' THEN 3 ELSE 4 END,
        t.created_at DESC
    `, [req.landlord.id]);
    log('info', 'Tickets fetched', { landlordId: req.landlord.id, count: rows.length });
    res.json(rows);
  } catch (err) {
    log('error', 'Failed to fetch tickets', { error: err.message });
    res.status(500).json({ error: 'Internal server error' });
  }
});

// POST /api/tickets/submit/:propertyId — Tenant submission (no auth required)
app.post('/api/tickets/submit/:propertyId', async (req, res) => {
  const { title, description, tenantName, tenantEmail, photoUrl, priority } = req.body;
  if (!title || !tenantName || !tenantEmail) {
    return res.status(400).json({ error: 'title, tenantName, and tenantEmail are required' });
  }
  const publicToken = crypto.randomBytes(32).toString('hex');
  try {
    const { rows } = await pool.query(`
      INSERT INTO maintenance_tickets 
        (property_id, title, description, photo_url, priority, public_token)
      VALUES ($1, $2, $3, $4, $5, $6)
      RETURNING id, public_token
    `, [req.params.propertyId, title, description, photoUrl, priority || 'medium', publicToken]);
    log('info', 'Ticket submitted', { ticketId: rows[0].id, propertyId: req.params.propertyId });
    res.status(201).json({ ticketId: rows[0].id, statusUrl: `/status/${rows[0].public_token}` });
  } catch (err) {
    log('error', 'Failed to submit ticket', { error: err.message });
    res.status(500).json({ error: 'Internal server error' });
  }
});

// PATCH /api/tickets/:id/assign — Assign contractor
app.patch('/api/tickets/:id/assign', authenticate, async (req, res) => {
  const { contractorId } = req.body;
  if (!contractorId) return res.status(400).json({ error: 'contractorId is required' });
  try {
    const { rows } = await pool.query(`
      UPDATE maintenance_tickets 
      SET contractor_id = $1, status = 'assigned'
      WHERE id = $2
      RETURNING *
    `, [contractorId, req.params.id]);
    if (!rows.length) return res.status(404).json({ error: 'Ticket not found' });
    log('info', 'Ticket assigned', { ticketId: req.params.id, contractorId });
    res.json(rows[0]);
  } catch (err) {
    log('error', 'Failed to assign ticket', { error: err.message });
    res.status(500).json({ error: 'Internal server error' });
  }
});

app.listen(3001, () => log('info', 'FixIt API running on port 3001'));
```

### 4.3 Main Frontend Component (`Dashboard.jsx`)

```jsx
// FixIt — Landlord Dashboard Component
// Generated by IBM Project Bob

import { useState, useEffect } from 'react';

const STATUS_COLORS = {
  open: 'bg-red-100 text-red-800',
  assigned: 'bg-yellow-100 text-yellow-800',
  in_progress: 'bg-blue-100 text-blue-800',
  resolved: 'bg-green-100 text-green-800',
};

const PRIORITY_COLORS = {
  emergency: 'border-l-4 border-red-600',
  high: 'border-l-4 border-orange-500',
  medium: 'border-l-4 border-yellow-400',
  low: 'border-l-4 border-gray-300',
};

export default function Dashboard() {
  const [tickets, setTickets] = useState([]);
  const [selected, setSelected] = useState(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    fetch('/api/tickets', {
      headers: { Authorization: `Bearer ${localStorage.getItem('token')}` }
    })
      .then(r => r.json())
      .then(data => { setTickets(data); setLoading(false); });
  }, []);

  const statusCounts = ['open', 'assigned', 'in_progress', 'resolved'].reduce((acc, s) => {
    acc[s] = tickets.filter(t => t.status === s).length;
    return acc;
  }, {});

  if (loading) return <div className="flex items-center justify-center h-screen">Loading...</div>;

  return (
    <div className="min-h-screen bg-gray-50">
      <header className="bg-white border-b px-6 py-4">
        <h1 className="text-xl font-semibold text-gray-900">FixIt — Maintenance Dashboard</h1>
      </header>
      <main className="max-w-5xl mx-auto px-6 py-6">
        {/* Status Summary Cards */}
        <div className="grid grid-cols-4 gap-4 mb-6">
          {Object.entries(statusCounts).map(([status, count]) => (
            <div key={status} className="bg-white rounded-lg border p-4">
              <p className="text-sm text-gray-500 capitalize">{status.replace('_', ' ')}</p>
              <p className="text-3xl font-bold text-gray-900 mt-1">{count}</p>
            </div>
          ))}
        </div>
        {/* Ticket List */}
        <div className="space-y-3">
          {tickets.map(ticket => (
            <div
              key={ticket.id}
              onClick={() => setSelected(ticket)}
              className={`bg-white rounded-lg border p-4 cursor-pointer hover:shadow-sm transition-shadow ${PRIORITY_COLORS[ticket.priority]}`}
            >
              <div className="flex items-start justify-between">
                <div>
                  <p className="font-medium text-gray-900">{ticket.title}</p>
                  <p className="text-sm text-gray-500 mt-1">{ticket.address} · {ticket.tenant_name}</p>
                </div>
                <span className={`text-xs px-2 py-1 rounded-full font-medium ${STATUS_COLORS[ticket.status]}`}>
                  {ticket.status.replace('_', ' ')}
                </span>
              </div>
            </div>
          ))}
        </div>
      </main>
    </div>
  );
}
```
