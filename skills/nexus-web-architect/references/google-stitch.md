# Google Stitch Reference

This document provides a guide to using Google Stitch for creating high-fidelity mockups.

## Accessing Google Stitch

Navigate to [stitch.withgoogle.com](https://stitch.withgoogle.com/) to begin.

## Writing Effective Prompts

Stitch understands natural language. To get the best results, provide specific details from your design system:

*   **Theme:** "dark theme", "light theme"
*   **Colors:** Use hex codes for primary, secondary, and accent colors.
*   **Typography:** Specify font names for headings and body text.
*   **Style:** Use keywords like "minimalist", "professional", "playful", "glassmorphism".

**Example Prompt:**

> "Design a dashboard for a project management tool. Use a light theme with a primary color of #4F46E5. The typography should be Inter. The style should be clean and modern, with cards for each project and a sidebar for navigation."

## Iterating and Refining

Once Stitch generates the initial design, you can use the built-in tools to:

*   **Directly edit text and components.**
*   **Change colors and fonts.**
*   **Rearrange elements on the canvas.**

Continue refining the design until it aligns with your vision.

## Exporting Your Work

Stitch offers several export options:

*   **Figma:** Export your design as a fully-editable Figma file, with layers and components intact.
*   **HTML/CSS:** Get clean, semantic HTML and CSS.
*   **Tailwind CSS:** Export your design with Tailwind CSS classes for easy integration into your project.
*   **React:** Generate React components with JSX and CSS Modules.
