# Ideal Customer Profile (ICP) Framework

This framework helps define the specific characteristics of a high-value enterprise account that is a strong fit for our solutions. A clear ICP ensures that sales and marketing efforts are focused on the accounts with the highest probability of closing.

## ICP Definition Table

Use the following table to build a detailed profile of your ideal customer. Score each attribute from 1 (low fit) to 5 (high fit).

| Attribute Category | Attribute | Description & High-Fit Indicators | Score (1-5) |
| :--- | :--- | :--- | :--- |
| **Firmographics** | Industry/Vertical | Specific industries where our solutions have a proven track record (e.g., Financial Services, Manufacturing, Utilities). | |
| | Annual Revenue | Typically > $1 Billion USD. | |
| | Employee Count | Typically > 5,000 employees. | |
| | Geographic Footprint | Global or multi-national presence. | |
| **Technographics** | Legacy Systems | Significant investment in legacy systems (e.g., Mainframes, Oracle, SAP, WebSphere). | |
| | Cloud Strategy | Publicly stated hybrid-cloud or multi-cloud strategy. | |
| | Key Vendors | Existing relationships with key partners (e.g., Red Hat, AWS, Azure). | |
| **Strategic Imperatives** | Digital Transformation | Publicly announced digital transformation, modernization, or innovation initiatives. | |
| | Cost Reduction Targets | Explicit goals to reduce operational or IT costs. | |
| | Compliance/Regulatory | Subject to significant regulatory pressure (e.g., NERC-CIP, EU AI Act, GDPR). | |
| **Pain Signals** | "Why Now?" Event | A clear compelling event, such as a pending license renewal, a recent security breach, or a major competitive threat. | |
| | Quantifiable Pain | The challenges can be directly tied to financial metrics (e.g., high maintenance costs, revenue loss from downtime). | |
| | C-Level Attention | The pain points are visible and discussed at the executive level. | |

## ICP Scoring

*   **Total Score > 50:** Tier 1 Account - High priority, immediate engagement.
*   **Total Score 35-50:** Tier 2 Account - Qualify further, nurture.
*   **Total Score < 35:** Tier 3 Account - Monitor, but do not actively pursue.
